# Vrane ‚Äì K-12  (MVP)

Vrane is a web app for **K-12 students** that helps them understand while reading.

A student:
1. Opens a reading passage.
2. **Highlights** any text (e.g. ‚Äúkindergarten through 12th grade‚Äù).
3. Immediately gets a **simple, kid-friendly explanation** generated by AI.

The goal is to let students learn at their own pace, while giving schools and teachers a tool that fits into existing platforms.

---

## üåü Main Features

- Web-based reading interface (works on laptop, tablet, Chromebook).
- Highlight any part of the text ‚Üí get a simple explanation.
- AI explanations tailored for **K-12 reading level**.

## üß± High-Level Architecture (Textual Diagram)

Below is a **plain-text architecture diagram** for how the system works:

```text
[ Student Browser (Web App) ]
          |
          | 1. Student opens passage and highlights text
          v
[ Frontend (React / Next.js / Vite) ]
          |
          | 2. Sends selected phrase + context via HTTPS (JSON)
          v

          v

          |
          | 7. Shows explanation panel under the highlighted text

```

## Step by Step ‚Äì workflow

Student opens the website and sees a reading passage.

The student highlights a word or phrase.

The browser (frontend) detects the selected text and sends it to the backend API.

The backend takes:

the selected phrase,

the full sentence/passage,the student grade level (if available),and builds a prompt for the OpenAI API.

The OpenAI model returns a short, simple explanation.

The backend sends that explanation back to the browser.

The browser shows the explanation in a panel below the text so the student can keep reading.

## Upcoming Teacher Dashboard Features

Expanding Vrane into a complete classroom and school-ready reading platform.

### 1. AI-Generated Individual Learning Plans (ILPs)
- Automatic weekly reports for each student.
- Highlights strengths, weaknesses, reading speed, and vocabulary gaps.
- Personalized recommendations for what to read next.

### 2. Difficulty Heatmaps for Passages
- Detect which phrases the entire class struggled with.
- Highlight high-confusion sections directly on the passage.
- Helps teachers adjust instruction and identify tough concepts.

### 3. Real-Time Class Monitoring (Live Mode)
- See which students are currently reading.
- Track highlights and difficulties in real time.
- Identify students who may be stuck or disengaged.

### 4. Assignment Builder + Auto-Generated AI Quizzes
- Teachers can upload their own reading material.
- Vrane automatically creates:
  - Vocabulary questions
  - Short answer questions
  - Summary prompts
- Saves teachers hours of prep time.

### 5. Reading Growth Timeline
- Tracks long-term growth in comprehension and vocabulary.
- Shows trends per student and per class.
- Predicts future reading performance.
- Helps administrators see measurable progress.
